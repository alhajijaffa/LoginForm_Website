## LoginForm Website (php Backend project)

A simple PHP backend login system built with PHP, HTML, CSS,JavaScript and MYSQL.
This project demostrate user authentication and role based access for Admin, Teacher and student

## Features
- User Login Authentication
- Admin Dashboard
- Teacher Dashboard
- Student Dashboard
- Logout functionality
- Responsive User Interface
- MySQL Database connection
- Organized project structure

## Technologies Used
* PHP
* MySQL
* HTML
* CSS
* JavaScript

  ## Installation
  
  1 clone the repository:
  
  Go to your terminal and create a new folder, inside your new folder
  type this => git clone https://github.com/alhajijaffa/LoginForm_Website.git

  2  Make sure you install the following installed:
     - PHP 8.x 
     - MySQL
     - A web server (Apache, php's built-in server)
       
  3 Create MySQL database

  4 Update the database credentials in config.php

  5 Start your web server

  6 open the project in your browser:
  ```
  https://localhost/LoginForm_Website/
 ```
  Or , if using PHP's built-in server:
 ```
  go to your terminal and type
  php-s localhost:800
  ```
